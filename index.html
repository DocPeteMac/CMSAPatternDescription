Description for the CMSA Pattern App

All CMSA pattern, with a different view, like up from the air, for a better understanding with an animated way how to ride and shoot the pattern!


This App is an educational App for beginner in the sport of mounted shooting as well as a reference for advanced shooter to look up some pattern. It contains the entire pattern and description regarding to the CMSA rulebook 2019-2020. All pattern are displayed in a 3 dimensional view for a better understanding of the pattern compare to black and white dots on plain paper. Another view is showing a path how to run the pattern. The path, how to run is a suggestion and is the simplest and most efficient way for beginner. It doesn't contain any variation for now. Variation on how to run certain pattern will be implemented in the future. On the screen how to shoot the pattern show in a simple animation weather you engage the targets straight or cross from your horse. The bird eye view is a different view for better understanding the particular pattern. The rang master view is showing the pattern similar to the rulebook on plain paper with documented distances as a 'pocket reference' to set up pattern correctly. Last but not least there is one view called >speed<, to show the beginner after he/she advances how to improve the times, where he/she can improve the speed while engaging the course - as a suggestion

Every CMSA pattern is available from the SELECT screen.

The PATTERN screen shows an overview over the pattern in 3D with correct relations.


The RUN screen shows the pattern in 3D with a suggestion on how to run the pattern.

The SHOOT screen show animated how to engage the pattern.

The RANGE MASTER screen show the pattern with exact detentions for better orientation or setup the pattern.

The DESCRIPTION window is available for every the pattern according to the rulebook.

The SPEED screen can help the beginner to improve his time while run the pattern.
